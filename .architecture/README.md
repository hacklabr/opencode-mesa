# Mesa Plugin — Architecture Decision Records

This directory contains the Architecture Decision Records (ADRs) for the Mesa plugin (v1.0.0). ADRs document the significant architectural decisions made during development, including the context that led to each decision, the decision itself, and its consequences.

ADRs are immutable once accepted — they serve as a historical record. If a decision is superseded, a new ADR is created that references the original.
